Calculate the height of bars in bar chart(s)

Proportionally rescales a set of values in relation to a single fixed value. (This isn't actually limited to bar chart heights).
Show prompt overview
Hide prompt overview

You
  1. Provide a list of values that represent elements in a data visualization (i..e., bars in a chart).
  2. Indicate the desired fixed size of the largest element.
GPT4
  1. Sets the largest element to the indicated size and scales the remaining elements proportionally.

Context

###

I am creating a bar chart for a website.

In this chart, the highest bar will always be set to the pixel height I specified under the heading 'Tallest bar', while the remaining bars will be proportionally scaled in relation to this tallest bar.###

Instructions

Review the bar heights provided under the heading 'Chart values' and calculate the pixel heights for each one in accordance with the context I provided. Additionally, if the calculated height for any bar falls below 10px, please highlight it as a potential issue.
