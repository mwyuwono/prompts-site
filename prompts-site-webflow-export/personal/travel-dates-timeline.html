Context:

"""I am planning a trip and would like your assistance to visualize these 'Destinations & dates' options on a gantt chart or timeline so that I can share them with my traveling companions and they will understand which destinations are available on which dates. The calendar will start on the earliest date provided and end on the latest date provided."""

Visualization Requirements

  1. Provide an interactive Gantt chart or timeline visualization of these events using matplotlib.
  2. Each event should be represented as a horizontal bar spanning its start and end dates, creating a 'swimlane' effect.
  3. Events should fill the height of their respective swim lanes, and the applicable dates for a given destination should be shaded for easy interpretability.
  4. Each event should occupy its own row.
  5. The shaded regions on each row should represent the duration of the event.
  6. The y-axis labels should be the names of the destinations; List the destination name on the top line and the 'City, State' on the next line.
  7. The x-axis should be formatted to show dates.
  8. Set the x-axis limits to cover the full duration from the earliest start date to the latest end date.
  9. The title of the chart should be 'Trip Options'.
  10. The background color of the chart should be #fff1e5
  11. The color of the bars should be #425052.
  12. There should be 24px padding around the visualization content for readability.
  13. Include the ability to download the visualization in .png format.
